<h2>Keyboard Shortcuts</h2>
<hr>

<b>All shortcuts: CTRL + ALT + key</b>

<ul>
<li>B → Base Tone</li>
<li>D → Dark & Moody</li>
<li>O → Teal & Orange</li>
<li>V → Vibrant</li>
<li>P → HDR</li>
<li>C → Switch profile</li>
<li>G → Settings / LUT / Filters</li>
<li>A → Auto-Scene-Match</li>
<li>I → Import / Export / Debug</li>
<li>S → Scopes HUD</li>
<li>X → GPU Mode</li>
<li>H → Show UI</li>
</ul>
